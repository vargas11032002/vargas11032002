### 🏢 JAV Dev Group

Soy el fundador de **JAV Dev Group**, una agencia de transformación digital enfocada en PYMES de Garzón y el sur del Huila. Ofrecemos desarrollo web, apps móviles, chatbots con IA, diseño gráfico y marketing digital — construyendo soluciones a la medida para negocios locales, agroindustria y organizaciones comunitarias.

**Servicios:**
- 💻 Desarrollo web (landing pages, e-commerce, plataformas a medida)
- 📱 Apps móviles (React Native / Expo)
- 🤖 Chatbots e integraciones con IA
- 🎨 Diseño gráfico y marca
- 📈 Marketing digital y SEO

### 🚀 Proyectos destacados

| Proyecto | Descripción | Stack |
|---|---|---|
| **SistemaJAC** | Plataforma multi-tenant para gestión de Juntas de Acción Comunal en Garzón: afiliados, tesorería, actas, portal de socios con carné digital animado | Next.js 14, Prisma 7, PostgreSQL, NextAuth |
| **StoreOS** | SaaS multi-tenant de punto de venta e inventario, con facturación electrónica DIAN offline-first y escaneo de código de barras multi-caja | Node.js, Prisma, PostgreSQL |
| **AgroSano Huila** | App móvil de detección de enfermedades en cultivos para agricultores rurales, con diagnósticos de IA vía Google Gemini 2.5 Flash | React Native, Expo, Gemini API |
| **CleanApp** | Plataforma de gestión de servicios para una empresa de aseo residencial en Garzón, con notificaciones automáticas por WhatsApp | Next.js, Express, Prisma/PostgreSQL, Twilio |
| **Garzón Digital** | Directorio comercial PWA con búsqueda en lenguaje natural potenciada por IA | Next.js |
| **Cebo El Arriero (AgroVida Orgánica)** | Landing page con checkout Wompi, notificaciones de pedidos por WhatsApp y ficha técnica descargable, más auditoría SEO completa | Next.js, Wompi, SEO técnico |
| **Mirador Turístico Conuco** | Landing premium de estética oscura con ilustraciones SVG a medida para un destino turístico respaldado por Fondo Emprender | Next.js, SVG a medida |
| **E-commerce Platform** | Plataforma de e-commerce propia de la agencia, con panel admin e inventario, pensada para escalar de tienda pequeña a supermercado | Next.js, PostgreSQL |
